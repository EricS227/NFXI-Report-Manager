# NFxi Report

Cash flow report generator with PDF export.

Sistema de fluxo de caixa com exportação em PDF.

## Quick Start / Início Rápido

```bash
pip install -r requirements.txt
python create_db_and_admin.py
python main.py
```

Access / Acesse: http://127.0.0.1:5000

**Login:** admin / changeme

## Features / Funcionalidades

- User authentication / Autenticação de usuários
- PDF reports by month/year / Relatórios PDF por mês/ano
- Transaction management / Gerenciamento de transações
- CSV import / Importação CSV

## Tech Stack / Tecnologias

Flask, SQLAlchemy, SQLite, ReportLab
